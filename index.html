Weekly

Weekly is a task calendar. You can add tasks according to what day you have to complete them, then click a toggle button to mark tasks complete. The tasks will be conveniently stored on your webpage to view at any time. Currently optimized for desktop user experience.

Goals

Have tasks you want to complete, but have not really set on what day you will complete it? Use Goals. Goals works like a traditional to-do list. Just type the name of the task, then add it to the list. Once you're finished, you can remove it. Made a typo in your task entry? No worries, you can edit it.
